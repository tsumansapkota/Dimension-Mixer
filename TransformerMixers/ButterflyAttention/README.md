## Butterfly Attention

This sub-directory includes code and experiments for Butterfly Attention.

### Benchmark

From the directory `TransformerMixers/ButterflyAttention/` run:   

```python benchmark_attention_butterfly.py --seed [SEED] --save_dir [SAVE DIRECTORY]```

By default, the `SEED` is `1` and `SAVE DIRECTORY` is `./logs` 

For Reproducibility Experiments use `SEED` from the list: `[147, 258, 369]` in each run.
